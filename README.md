# Vaccine Schedule Fortaleza

Is a django project to find out if your name is in Fortaleza schedule list to get vaccineted.

# Requirements

- Docker 19.03.6+
- Compose 1.24.1+

## Setup
    
```bash
docker-compose -f docker-compose.dev.yml up --build --remove-orphans
```
